# 🎮 超级华容道 (Super Huarong Road) - AI 旗舰版

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg) ![License](https://img.shields.io/badge/license-MIT-green.svg)

**超级华容道** 是一款基于 Web 技术构建的单文件益智游戏。本项目完美复刻了经典电子游戏机的外观与手感，并内置了强大的 **BFS (广度优先搜索) AI 求解器**，能够针对任意局面实时计算最优解，实现真正的智能提示。

---

## ✨ 核心亮点 (Key Features)

### 🤖 真·AI 智能求解
不同于传统的“预设步骤”死板提示，本项目内置了完整的求解算法引擎：
* **实时计算**：无论棋盘被用户走得多么混乱，点击“提示”时，AI 会基于当前局面实时计算最短路径。
* **状态识别**：自动判断当前是否为死局，并给出相应反馈。
* **智能演示**：AI 接管控制权，自动移动棋子演示下一步最佳走法。

### 🧩 500+ 逻辑关卡
* **经典基底**：内置 12 个经过数学验证的经典布局（涵盖“兵临城下”、“横刀立马”、“层层设防”等）。
* **算法扩展**：通过镜像翻转、ID 轮换等算法，将基础关卡动态扩展至 500+ 个不重复的挑战。
* **难度平滑**：从新手入门到宗师级别，难度循序渐进。

### 🎨 极致的 UI/UX 体验
* **拟物化设计**：深蓝色磨砂质感机身，配合 LED 点阵风格的计步器与关卡显示。
* **视觉修正**：红色方块上的 **"KK" Logo** 采用了特殊的 CSS 变换 (`translate(-50%, -55%)`)，完美修正了字体的视觉重心下沉问题，确保绝对居中。
* **自适应布局**：完美支持 PC 鼠标拖拽与手机触屏滑动，棋盘始终保持 4:5 黄金比例。

### 🔊 沉浸式音效
* **零素材依赖**：不加载任何外部 MP3/WAV 文件，完全使用 **Web Audio API** 实时合成音效。
* **听觉反馈**：包含清脆的棋子滑动声、边界撞击声以及通关时的 8-bit 胜利旋律。

## 🚀 快速开始 (Quick Start)

本项目采用 **单文件 (Single-file)** 架构，无需 Node.js、Webpack 或数据库环境。

1.  **下载**：获取本项目中的 `index.html` 文件。
2.  **运行**：双击文件，使用 Chrome / Edge / Safari / Firefox 等现代浏览器打开即可。
3.  **开始挑战**：
    * **移动**：拖动或滑动棋子，将红色的 **KK** 方块移至底部出口。
    * **重置**：点击红色按钮重新开始当前关卡。
    * **提示**：卡关时点击黄色 **提示 (Hint)** 按钮，观看 AI 演示。

## 📦 部署指南 (Deployment)

由于是纯静态文件，您可以将其部署到任何 Web 服务器或对象存储中。

### ☁️ 云服务器 (Nginx/Apache)
将 `index.html` 上传至网站根目录（例如 `/var/www/html/`），直接访问域名/IP即可。

### ⚡ 静态托管 (Netlify / GitHub Pages)
* **Netlify Drop**: 直接将包含 `index.html` 的文件夹拖入 Netlify 部署页面。
* **GitHub Pages**: 上传至仓库，开启 Pages 服务即可。

## 🛠️ 技术栈 (Tech Stack)

* **HTML5**: 语义化结构。
* **CSS3**: Flexbox/Grid 布局, CSS Variables, `clamp()` 响应式排版。
* **JavaScript (ES6+)**:
    * **BFS Algorithm**: 广度优先搜索算法，用于路径规划。
    * **State Compression**: 状态压缩与去重，优化搜索性能。
    * **Web Audio API**: 音频合成。
